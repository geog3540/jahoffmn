Animal Agriculture Parallel Coordinates Plot

The map you're looking at displays the Iowa's animal agriculture distributions in a multivariate manner. This allows for users to investigate which types of animals are most common in each county, as well as how the compare across the state.

  • P.C. enables you (the user) an overview of all the data.
  • There are three primary points of interaction; brushing P.C. axes, the dropdown attribute selection, and the map hovering. Each of them aid in assisting the user to get the data they need by zooming and filtering the data.
  • The map tooltip interaction allows precise information, thus serving as a proxy for details on demand.
